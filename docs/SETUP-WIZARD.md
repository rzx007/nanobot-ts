# 配置向导（Setup Wizard）

配置向导在首次运行或未检测到有效配置时引导用户完成基础配置，包括模型供应商、安全设置与消息渠道。

## 向导步骤总览

| 步骤 | 标题     | 说明                 |
|------|----------|----------------------|
| 0    | Provider 配置 | 选择并配置 AI 模型提供商 |
| 1    | 安全设置 | 配置工具确认机制     |
| 2    | 渠道配置 | 配置消息渠道（可选） |

## ProviderForm 操作步骤

第一步「Provider 配置」按以下顺序进行：

1. **先列出供应商**  
   展示供应商列表（OpenAI、Anthropic、OpenRouter、DeepSeek、Groq、Google），列表**最后一项**为「跳过此步骤」。

2. **选中供应商**  
   使用 ↑/↓ 在列表中移动，Enter 确认当前高亮项。  
   - 若选择某一供应商，进入步骤 3、4。  
   - 若选择「跳过此步骤」，则本步不配置供应商/API Key/模型，直接可进入下一步；后续在使用时再设置（见下文）。

3. **粘贴 API Key 并确认**  
   选择具体供应商后：  
   - 「如何提供 API Key？」下可选「粘贴 API Key（存入配置文件）」或「暂不填写」。  
   - 选「粘贴」则显示输入框，用户粘贴并确认；选「暂不填写」则跳过 API Key。  
   - 若本步选择了「跳过此步骤」，则不展示 API Key 与默认模型，无此环节。

4. **默认模型**  
   在 API Key 确认之后，选择或输入默认模型（可从常用模型列表选或手动输入）。  
   若已选择「跳过此步骤」，则无默认模型设置环节。

### 跳过此步骤

- 供应商列表最后一项为 **「跳过此步骤」**。选中并确认后，本向导中不配置模型与供应商，保存时可不写入或仅写入空的 provider 结构。
- **后续在使用时**可通过配置界面（如 `/config`）或再次进入配置向导，再设置或更换模型与供应商。

## 键盘约定

- **↑/↓**：在垂直列表（供应商、渠道、导航项）中移动焦点。
- **←/→**：二选一（如是/否、启用/禁用）时切换选项。
- **Enter / Space**：确认当前选项或进入下一子步骤。
- **Escape**：取消当前编辑，或返回上一步 / 退出向导。
- **Tab**：在字段间切换（如提供商 / API Key / 模型）。

## 后续可再设置

使用阶段可在配置界面或命令中再次设置/更换模型与供应商，首次 setup 可只完成必要项或选择「跳过此步骤」稍后再配。
